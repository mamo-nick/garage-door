# Otevírání garážových vrat pomocí wi-fi relé
Implementace wi-fi relé pro ovládání garážových vrat do stávajícího řešení a jeho integrace do Home assistant.

## Stávající řešení
Rolovací garážová vrata ovládaná jednostavovým relé. 
Je to klasické řešení, kdy je relé použito pro všechny stavy. Každým sepnutím relé způsobuji nasledující stav: Nahoru - Vypnout - Dolů - Vypnout.
